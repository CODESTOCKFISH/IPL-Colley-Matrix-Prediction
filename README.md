# IPL Colley Matrix Prediction

## 📌 Overview
This project ranks IPL teams and predicts match outcomes using the Colley Matrix method (linear algebra).

## 🧠 Methodology
- Construct matrix C and vector b from match results
- Solve Cr = b to get team ratings
- Combine 2025 and 2026 data (70-30 weighting)
- Predict remaining matches using:
  - Home advantage
  - Toss advantage
  - Tie handling

## 📂 Dataset
The file `matches.csv` contains IPL match data.

## ⚙️ Tech Stack
- Python
- NumPy
- Pandas

## 📊 Output
- Team rankings
- Match predictions
- Final standings

## 📉 Limitations
- No player-level data
- Simplified assumptions
